Piskel Clone

Piskel is a free online editor for animated sprites & pixel art. It allows to create animations right in your browser.

This application is a clone of Piskel App, created for The Rolling Scopes School.

Available tools:

  • - Pen
  • - Paint Bucket
  • - Paint Same Pixels
  • - Circle
  • - Rectangle
  • - Eraser
  • - Line
  • - Triangle
  • - Rhombus
  • - Lighten
  • - Darken
  • - Dithering
  • - Color Picker
  • - Semicircles

Available functionality:

  • - You can add new frames, delete
    frames or clone existing frames

  • - You can preview your result
    and open it in fullscreen

  • - You can control
    Frames Per Second number
